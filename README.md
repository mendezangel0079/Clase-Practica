# Clase-Práctica - Equipo 3

##API de Comparación de Precios (MockAPI + Postman)

Este repositorio contiene una práctica para estudiantes de primer semestre relacionada con Fundamentos de Ingeniería de Software, donde se crea una API falsa (mock) utilizando MockAPI y se consume desde Postman Web. Esta clase no necesita de la instalación de algún programa.

La API permite almacenar, consultar, editar y eliminar productos de supermercado para comparar precios.

-Objetivo de la Práctica
  -Comprender qué es una API.
  -Crear un recurso usando MockAPI.
  -Realizar operaciones CRUD con Postman Web (GET, POST, PUT, DELETE).
  -Manipular datos falsos (fake data) para simular una base de datos real.

*Herramientas Utilizadas
  -MockAPI (https://mockapi.io) → Para crear la API fake.
  -Postman Web (https://web.postman.co) → Para consumir la API.
  -JSON → Formato utilizado para enviar y recibir datos.

#Práctica
##Pasos

1. Entra a https://mockapi.io.
2. Crea un nuevo Project con el nombre de tu elección
3. En API PREFIX poner “ /api/v1 “ y crea tu proyecto
4. Crea un Resource llamado exactamente: Productos
5. Click en la X del lado derecho de los bloques dejando únicamente la fila de ID

![Schema del proyecto en mockapi](https://github.com/user-attachments/assets/11dad8d4-f028-453d-983f-2a1f4aadc8e1)

6. Crear nuevos elementos en Schema (campos del producto)
7. Clic en Add field (botón azul)

![Boton para agregar nuevos elementos](https://github.com/user-attachments/assets/e805389d-a164-43f5-888d-734f203563cd)

Agrega los siguientes elementos:
  -Name
  -Price
  -Stock
